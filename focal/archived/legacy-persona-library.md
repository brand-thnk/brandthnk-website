## AutomatedPersonaSelector Algorithm (Legacy)

The original system used a sophisticated weighted matching algorithm:

```javascript
class AutomatedPersonaSelector {
  constructor() {
    this.weights = {
      organizationType: 0.4,    // Primary organization alignment
      targetAudience: 0.3,      // Audience relevance
      marketType: 0.2,          // Geographic/market fit
      contentAnalysis: 0.1      // Message content signals
    };
  }

  selectPersonas(businessParams) {
    const primarySegment = this.determinePrimarySegment(businessParams);
    const allPersonas = this.getAllPersonas();

    })).sort((a, b) => b.score - a.score).slice(0, 4);
  }

  determinePrimarySegment(businessParams) {
    if (businessParams.organizationType === 'fintech' ||
        businessParams.targetAudience === 'fintech-buyers') {
      return 'fintech';
    }
    if (businessParams.targetAudience === 'business-customers') {
      return 'business';
    }
    return 'consumer';
  }
}
```

### Matching Criteria Framework

Each persona included detailed `autoMatchCriteria`:

- **Market Type Alignment:** Geographic and demographic scoring factors

## Research Methodology (Legacy)

### Three-Phase Interview Framework

**Phase 1: Context Setting (Turns 1-3)**
- Persona establishes current situation and primary challenges
- Natural conversation building trust and understanding
- No messaging introduction yet

**Phase 2: Messaging Reaction (Turns 4-8)**
- Specific messaging and value propositions presented
- Authentic persona reactions based on documented patterns
- Core research insights captured

**Phase 3: Deep Probe (Turns 9-11)**
- Deeper exploration of Phase 2 reactions and underlying drivers
- Connection to core motivations and decision criteria
- Strategic insights for messaging refinement

### Cross-Persona Synthesis

The legacy system included sophisticated synthesis across selected personas:
- **Agreement Patterns:** Where multiple personas aligned on messaging
- **Divergent Reactions:** Key differences in persona responses
- **Strategic Gaps:** Missing elements identified across persona feedback
- **Implementation Priorities:** Ranked recommendations based on persona importance
